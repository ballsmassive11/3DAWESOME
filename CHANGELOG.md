# Changelog

All notable changes to this project will be documented in this file.

## [0.5.0] - 2026-04-30
### Added
- New "Guy" entity that wanders randomly in terrain chunks.
- Entity distance culling for performance.
- NPC movement logic to avoid water.

## [0.4.1] - 2026-04-30
### Added
- New biomes: Forest, Tundra, Desert, Stony Peaks, and Meadows.
- Height-based biome distribution.

## [0.4.0] - 2026-04-28
### Added
- Procedural world generation improvements.

## [0.3.1] - 2026-04-26
